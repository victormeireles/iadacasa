Pill-shaped primary action control — use for any clickable action; the green `primary` variant is the default CTA, `accent` (mustard) for highlight moments, `secondary`/`ghost` for lower emphasis.

```jsx
<Button variant="primary" size="lg" rightIcon={<i data-lucide="arrow-right" />}>
  Montar meu sistema
</Button>
```

Variants: `primary` (green), `accent` (mustard, dark text), `secondary` (bordered paper), `ghost` (text-only green), `danger` (wine).
Sizes: `sm` / `md` / `lg`. Props: `leftIcon`, `rightIcon`, `fullWidth`, `disabled`, `href` (renders an `<a>`).
