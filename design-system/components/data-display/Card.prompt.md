The primary surface container — warm paper, hairline border, soft shadow, generous rounding. Wrap any grouped content.

```jsx
<Card padding="lg" interactive>
  <h3>Ficha Técnica</h3>
  <p>Padronize fichas e calcule o custo de cada prato.</p>
</Card>
```

Props: `padding` (none/sm/md/lg), `elevation` (none/sm/md), `interactive` (hover-lift), `inverse` (deep-green feature panel), `as`.
